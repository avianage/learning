# Java Introduction & Setup — Quick Reference

## What Is Java?
General-purpose, OOP language (Gosling, Sun Microsystems, 1995; Oracle since 2010).
**WORA** — Write Once, Run Anywhere: compiled code runs unchanged on any OS.

## Why It Matters
- Dominates enterprise backends (banking, insurance, e-commerce)
- Android apps (Java/Kotlin, same runtime)
- Huge ecosystem, strong typing → good for large/long-lived systems
- Java 8 (2014) added functional programming — major turning point

## JDK vs JRE vs JVM
```
JDK (Java Development Kit)          — what developers install
├── javac      → compiler (.java → .class)
├── java       → launcher (starts JVM)
├── javadoc    → generates docs
├── jar        → packages classes into .jar
└── JRE (Java Runtime Environment)  — what end users need to run Java
     └── JVM (Java Virtual Machine) — executes bytecode, platform-specific
          ├── Class Loader        → loads .class files
          ├── Bytecode Verifier   → security check
          └── Execution Engine
               ├── Interpreter   → line-by-line execution
               └── JIT Compiler  → compiles hot code to native
```
| Term | One-liner |
|------|-----------|
| JDK | Everything to write, compile, run Java |
| JRE | JVM + libraries, no compiler (not distributed separately since Java 11) |
| JVM | Executes bytecode; platform-specific, same bytecode everywhere |

## How Code Runs
```
Employee.java  --javac-->  Employee.class (bytecode)  --JVM-->  Output
```
`.class` = bytecode for an imaginary machine (the JVM), **not** native CPU code. Each platform's JVM translates it to native instructions — this is WORA.

## Java Versions
| Version | Year | Why It Matters |
|---------|------|-----------------|
| 1.0 | 1996 | Origin |
| 5 | 2004 | Generics, enums, for-each |
| **8** | **2014** | **Lambdas, Streams, Optional — courseware baseline** |
| 11 | 2018 | First LTS after 8, HttpClient |
| 17 | 2021 | Records, sealed classes |
| 21 | 2023 | Virtual threads — current LTS |

**LTS** = Long Term Support (extended updates). Non-LTS = stepping stones.
> This courseware baseline is Java 8; modern features (11–24) covered separately.

## Setup — 3 Steps
```bash
# 1. Install JDK 21 (LTS) from https://adoptium.net

# 2. Verify
java -version     # openjdk 21.0.x...
javac -version    # javac 21.0.x

# 3. Install an IDE (optional but recommended)
#    IntelliJ IDEA Community  → jetbrains.com/idea/download
#    Eclipse                  → eclipse.org/downloads
```
> JDK 21 runs Java 8 code fine and unlocks features up to Java 21.

## First Program
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
```bash
javac HelloWorld.java   # → HelloWorld.class
java HelloWorld         # → Hello, World!
```
